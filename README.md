# KeepGoing Releases

Pre-built binaries and installers for [KeepGoing](https://keepgoing.dev) desktop apps.

## Downloads

Head to the [Releases](https://github.com/keepgoing-dev/releases/releases) page to download the latest versions.

### Desktop Tray (macOS)

System tray companion that surfaces re-entry briefings from the menubar. Works with any editor.

**Install:** Download the `.dmg` from the [latest release](https://github.com/keepgoing-dev/releases/releases/latest).

> **macOS Gatekeeper:** This app is not code-signed. On first launch, right-click the app and select "Open" to bypass Gatekeeper.

## Other install methods

| App | Install |
|-----|---------|
| **VS Code Extension** | [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=keepgoing.keepgoing) |
| **MCP Server** | `npm install -g @keepgoingdev/mcp-server` |
| **CLI** | `npm install -g @keepgoingdev/cli` |
